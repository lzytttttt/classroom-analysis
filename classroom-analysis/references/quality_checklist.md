# 课堂分析报告质量检查清单

本清单对应 SKILL.md 的 15 段输出模板，逐段设置检查项。完成报告后逐段核对。

---

## 第 1 段：课堂基本判断

- [ ] 学科、学段、课型、总时长、核心学习任务五项均已填写
- [ ] 每项判断都附有依据（具体时间、文本片段或上下文信息）
- [ ] 无法推断的项目标注了"材料未显示"并说明需要什么证据
- [ ] 核心学习任务描述具体，不笼统（如"学习分数"应为"理解分数的意义"）

## 第 2 段：课堂环节切分

- [ ] 环节切分基于实际教学逻辑，而非机械等分时间
- [ ] 每个环节标注了教师主要行为和学生主要行为
- [ ] 每个环节识别了学习任务和可观察的学习证据
- [ ] 环节之间的过渡和衔接有描述
- [ ] 如无时间戳，使用"环节序号"而非具体时间

## 第 3 段：教师行为分析

- [ ] 教师行为分类有具体片段支撑，非泛泛而谈
- [ ] 分析了行为对学习的作用（创造了什么学习机会）
- [ ] 指出了可能的风险或局限
- [ ] 关键教学行为的解读包含"证据—分析—教研价值"三层
- [ ] 避免了对教师个人的评价，聚焦于教学行为

## 第 4 段：学生学习行为分析

- [ ] 区分了"学生在场"和"学生做了有意义的学习行为"
- [ ] 每类学生行为有具体时间/片段依据
- [ ] 标注了证据强度（强/中/弱/无）
- [ ] 关注了不同学生的差异化表现（非仅"全班"）
- [ ] 分析了学生行为的学习意义

## 第 5 段：问题链与互动分析

- [ ] 梳理了教师的主要问题及其学生回应
- [ ] 对每个问题标注了问题类型和认知层级
- [ ] 分析了追问/反馈的质量
- [ ] 识别了"接住学生回答"和"未充分展开"的片段
- [ ] 评估了问题链的递进性
- [ ] 提供了可改进的追问示例

## 第 6 段：认知层级分析

- [ ] 按环节/任务标注了主要认知层级
- [ ] 判断依据来自学生实际回应，而非仅教师提问
- [ ] 结合学段判断认知层级是否合理（参考 cognitive_levels.md）
- [ ] 分析了是否存在层级递进
- [ ] 指出了可提升认知层级的空间

## 第 7 段：学习证据分析

- [ ] 从理解、应用、迁移、表达、修正、产出六个维度收集证据
- [ ] 每项证据标注了证据强度（强/中/弱/无）
- [ ] 明确了证据可以支持什么学习判断
- [ ] 指出了证据的局限性
- [ ] 区分了"教师意图"和"学生实际证据"

## 第 8 段：课堂节奏分析

- [ ] 有时间戳时分析了各环节时长和比例
- [ ] 无时间戳时分析了环节密度和顺序合理性
- [ ] 判断了重要任务是否获得了充足时间
- [ ] 识别了高认知任务被赶进度或低认知任务耗时过多的问题
- [ ] 给出了具体的时间调整建议

## 第 9 段：目标—活动—评价一致性分析

- [ ] 识别了学习目标（显性或隐性）
- [ ] 分析了活动是否直接服务于核心目标
- [ ] 分析了是否有证据表明学生达成了目标
- [ ] 判断了反馈是否针对目标而非仅管理正确率/参与度
- [ ] 指出了目标、活动、评价之间的脱节

## 第 10 段：课堂亮点

- [ ] 每个亮点有具体证据（时间、片段）
- [ ] 分析了亮点的教学价值
- [ ] 提炼了可迁移的教研观点
- [ ] 亮点数量适当（2-3 个），非面面俱到

## 第 11 段：问题诊断

- [ ] 每个问题有具体证据
- [ ] 诊断聚焦于设计/行为问题，非个人批评
- [ ] 分析了问题对学生学习的具体影响
- [ ] 将问题转化为可研究的表述
- [ ] 问题数量适当（2-3 个），非罗列所有不足

## 第 12 段：教学改进建议

- [ ] 每条建议对应一个具体诊断出的问题
- [ ] 建议具体、可操作（非"增加互动"之类的泛泛建议）
- [ ] 包含可直接使用的课堂话语、任务设计或反馈语
- [ ] 说明了调整后预期出现什么学习证据
- [ ] 建议数量与问题数量匹配

## 第 13 段：可研发文章选题

- [ ] 选题基于课堂实际发现，非凭空想象
- [ ] 每个选题有问题意识和核心论点
- [ ] 标注了支撑选题的课堂证据
- [ ] 推荐了研究角度和文章类型
- [ ] 选题具有可论证性（非仅描述性标题）

## 第 14 段：推荐文章大纲

- [ ] 推荐理由说明了为什么这是最佳选题
- [ ] 中心论点明确、可论证
- [ ] 大纲结构对应所选文章类型（参考 article_templates.md）
- [ ] 每个部分标注了核心内容和写作要点
- [ ] 大纲可直接用于写作

## 第 15 段：分析限制说明

- [ ] 列出了材料中缺失的信息
- [ ] 说明了因缺失而不能确认的判断
- [ ] 建议了需要补充的证据类型
- [ ] 语言诚实、不回避局限

---

## 全局检查

- [ ] 全文遵循"证据优先"原则：每个重要判断都有依据
- [ ] 避免了"气氛活跃""效果良好"等无证据的空泛评价
- [ ] 未对教师进行人格评价，聚焦教学行为和设计
- [ ] 引用了具体时间范围或课堂片段
- [ ] 不确定的判断使用了审慎措辞（"文本显示""可初步判断""尚不能证明"）
- [ ] 文章选题的问题意识和核心论点可论证
- [ ] 改进建议与诊断出的问题一一对应
