# 教研文章选题模板

本文档为六种常见教研文章类型提供结构化大纲模板。在 SKILL.md 第 14 段生成文章选题和大纲时，先确定文章类型，再使用对应模板。

---

## 一、案例分析

**定位**：选取一个课堂片段或教学事件，进行深入剖析，揭示教学原理。

**适用场景**：有一个精彩的或值得反思的教学片段。

```
标题：[具体现象] + [教学启示]
  例：从一次"冷场"到"思维涌动"——追问策略在概念教学中的运用

一、案例描述
  - 教学背景（学科、学段、课题、学情）
  - 关键片段（完整呈现师生对话或教学行为）

二、案例分析
  - 现象解读：这个片段中发生了什么？
  - 理论透视：用什么理论/原理解释这个现象？
  - 多角度分析：从教师行为、学生反应、任务设计等角度分析

三、教学启示
  - 从案例中提炼的教学原则或策略
  - 可迁移的建议

四、结语
  - 一句话总结核心观点
```

**写作要点**：案例要具体、真实；分析要有理论深度而非仅描述现象；启示要可操作。

## 二、教学反思

**定位**：教师对自己的一节课进行回顾、分析和再设计。

**适用场景**：教师上完一节课后的自我反思，特别是公开课、研究课后的深度反思。

```
标题：[反思主题]——[课题]教学反思
  例：让学习真正发生——"分数的意义"教学反思

一、教学背景
  - 课前设计意图
  - 学情分析
  - 教学目标

二、课堂回放
  - 关键环节的实际教学过程
  - 预设与生成的差异

三、反思与分析
  - 成功之处：哪些设计达成了预期目标？证据是什么？
  - 不足之处：哪些环节效果不佳？原因是什么？
  - 意外收获：有哪些非预设的精彩生成？
  - 遗憾之处：有哪些教学机会被错过？

四、再设计
  - 如果重新上这节课，会如何调整？
  - 具体的改进方案

五、延伸思考
  - 这节课引发的更深层的教学思考
```

**写作要点**：要有真实的课堂细节；反思要深入到教学原理层面；再设计要具体可行。

## 三、课例研究报告

**定位**：基于"设计—实施—观察—改进"循环的系统性课例研究。

**适用场景**：教研组围绕一节课进行多轮磨课、集体观察和改进。

```
标题：[研究主题]——[课题]课例研究
  例：指向深度学习的问题链设计——"勾股定理"课例研究

一、研究背景与问题
  - 教研主题的确定
  - 研究的问题是什么
  - 相关文献或理论依据

二、教学设计
  - 教材分析与学情分析
  - 教学目标
  - 教学流程与设计意图
  - 重点环节的设计说明

三、课堂观察
  - 观察维度和观察工具
  - 观察数据（片段记录、统计数据）
  - 关键发现

四、分析与讨论
  - 数据分析：观察数据说明了什么？
  - 问题归因：效果不佳的环节原因是什么？
  - 成功经验：效果好的环节做对了什么？

五、改进与再实践
  - 基于分析的改进方案
  - 再次实施的效果
  - 前后对比

六、研究结论
  - 提炼的教学策略或原则
  - 可推广的经验
  - 后续研究方向
```

**写作要点**：要有观察数据支撑；分析要基于证据而非印象；结论要有可迁移性。

## 四、课堂观察报告

**定位**：基于课堂观察工具和数据的客观分析报告。

**适用场景**：听评课活动、教研组集体观课、教学评比。

```
标题：[观察维度]视角下的[课题]课堂观察
  例：学习证据视角下的"光的折射"课堂观察

一、观察背景
  - 授课教师、学科、学段、课题
  - 观察维度的选择及理由
  - 观察工具（观察量表、编码体系等）

二、观察发现
  2.1 [维度一]观察结果
    - 数据呈现（表格、图表）
    - 数据解读
  2.2 [维度二]观察结果
    - 数据呈现
    - 数据解读

三、问题分析
  - 从数据中发现的主要问题
  - 问题的可能原因
  - 对学生学习的影响

四、改进建议
  - 针对发现的问题的具体建议
  - 可操作的改进措施

五、观察反思
  - 观察工具的适用性
  - 观察方法的局限
  - 后续观察建议
```

**写作要点**：数据要客观、可验证；分析要区分事实和推断；建议要基于数据分析。

## 五、行动研究报告

**定位**：教师针对教学问题，设计并实施改进方案，报告行动过程和效果。

**适用场景**：教师在日常教学中发现问题并持续改进。

```
标题：[改进策略]促进[学习效果]的行动研究
  例：以"学习单"促进初中生数学自主预习的行动研究

一、问题的提出
  - 发现了什么教学问题
  - 问题的具体表现
  - 为什么这个问题重要

二、方案设计
  - 文献综述：已有的研究和经验
  - 理论依据
  - 改进方案的设计
  - 预期目标

三、行动实施
  第一轮行动
    - 实施过程
    - 数据收集（学生作业、测试、访谈、观察等）
    - 效果分析
    - 问题与调整

  第二轮行动
    - 调整后的方案
    - 实施过程
    - 效果分析

四、研究结论
  - 改进方案是否有效
  - 有效的机制是什么
  - 局限与不足

五、反思与展望
  - 行动过程中的收获
  - 后续改进方向
```

**写作要点**：问题要真实、具体；行动过程要详细记录；效果要有数据支撑。

## 六、教学策略论文

**定位**：从教学实践中提炼教学策略，结合理论进行论证。

**适用场景**：总结多年教学经验或系列研究，形成系统性策略。

```
标题：[策略名称]在[学科/领域]教学中的实践与思考
  例：以"问题链"驱动初中物理概念教学的策略探究

一、引言
  - 为什么要研究这个策略
  - 当前教学中存在的问题
  - 研究的意义

二、理论基础
  - 相关教育理论
  - 已有研究成果
  - 理论框架

三、策略构建
  3.1 策略的内涵与特征
  3.2 策略的实施原则
  3.3 策略的操作步骤
    - 步骤一：[名称]——[具体做法]
    - 步骤二：[名称]——[具体做法]
    - 步骤三：[名称]——[具体做法]

四、实践案例
  - 案例一：[简要描述+效果]
  - 案例二：[简要描述+效果]

五、效果分析
  - 学生学习效果的变化
  - 教学行为的变化
  - 数据支撑

六、讨论与建议
  - 策略的适用条件
  - 可能的局限
  - 实施建议

七、结语
  - 核心观点
  - 未来展望
```

**写作要点**：策略要有理论依据；操作步骤要具体可执行；效果要有实证支撑。

## 选题匹配指南

| 课堂分析发现 | 推荐文章类型 | 理由 |
|---|---|---|
| 有一个精彩的追问片段 | 案例分析 | 适合深入剖析单一事件 |
| 教师对自己的课不满意 | 教学反思 | 适合自我审视和再设计 |
| 教研组磨课多轮改进 | 课例研究报告 | 适合呈现研究过程 |
| 有系统的观察数据 | 课堂观察报告 | 适合数据驱动的分析 |
| 发现问题并持续改进 | 行动研究报告 | 适合记录改进过程 |
| 总结系列实践经验 | 教学策略论文 | 适合系统性策略提炼 |
