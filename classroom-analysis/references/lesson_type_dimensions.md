# 课型特化分析维度

本文档为不同课型提供额外的分析关注点。在使用 classroom-analysis skill 的 15 段通用分析框架时，根据课型叠加以下特化维度。

---

## 新授课

**核心关注**：概念建构过程、学生从"不知到知"的证据、先验知识的激活方式。

| 特化维度 | 观察要点 |
|---|---|
| 情境导入 | 是否创设了有意义的问题情境？情境与新知的关联是否清晰？ |
| 先验激活 | 教师如何唤起已有知识？是否建立了新旧知识的桥梁？ |
| 概念呈现 | 新概念/新知识是如何呈现的？是直接告知还是引导发现？ |
| 初步理解 | 学生是否表现出对新知的初步理解？证据是什么？ |
| 变式应用 | 是否提供了不同情境下的应用机会？ |

**易犯错误**：导入过长挤压新知建构时间；先验激活流于形式；缺少从具体到抽象的过渡。

---

## 复习课

**核心关注**：知识结构化、错题利用、变式训练、综合应用。

| 特化维度 | 观察要点 |
|---|---|
| 知识梳理 | 是教师罗列还是学生自主建构知识网络？ |
| 错题分析 | 是否利用典型错误？学生是否分析错因而非仅改正答案？ |
| 变式设计 | 变式是否覆盖核心概念的不同侧面？是否促进深层理解？ |
| 综合应用 | 是否有跨知识点的综合任务？ |
| 查漏补缺 | 是否针对薄弱环节进行了强化？ |

**易犯错误**：简单重复新授课内容；变成习题讲评课；缺少知识结构化环节。

---

## 实验课

**核心关注**：操作规范、探究过程、数据记录、科学思维。

| 特化维度 | 观察要点 |
|---|---|
| 实验设计 | 学生是否理解实验目的？是否参与方案设计？ |
| 操作规范 | 操作是否规范安全？教师是否示范关键步骤？ |
| 观察记录 | 学生是否有系统的观察和记录？ |
| 数据分析 | 学生如何处理数据？是否进行比较、归纳、推理？ |
| 结论提炼 | 结论是学生得出还是教师直接给出？ |
| 安全意识 | 安全提示和应急处理是否到位？ |

**易犯错误**：学生照方抓药式操作缺少思考；数据记录流于形式；结论提炼仓促。

---

## 写作课

**核心关注**：审题指导、构思过程、修改反馈、范文使用。

| 特化维度 | 观察要点 |
|---|---|
| 审题训练 | 是否有明确的审题指导？学生是否学会了分析题意？ |
| 素材激活 | 如何帮助学生打开思路、积累素材？ |
| 构思指导 | 是否有列提纲、组织结构的指导？ |
| 范文使用 | 范文是用于模仿还是启发？是否分析了写法？ |
| 写作实践 | 课堂是否有充分的写作时间？ |
| 修改反馈 | 反馈是否具体？是否指导学生如何修改？是否有同伴互评？ |

**易犯错误**：审题指导缺失；范文变成模板；反馈笼统（如"写得不错"）；缺少修改环节。

---

## 阅读课

**核心关注**：阅读策略、文本解读深度、批注习惯、多元理解。

| 特化维度 | 观察要点 |
|---|---|
| 阅读策略 | 是否显性教授了阅读策略（预测、提问、可视化、概括、推断）？ |
| 文本解读 | 解读停留在字面意思还是深入到深层含义？ |
| 批注习惯 | 学生是否有批注、圈画、做笔记的行为？ |
| 多元理解 | 是否鼓励不同解读？是否尊重学生个性化理解？ |
| 读写联结 | 阅读是否与表达、写作产生联结？ |

**易犯错误**：以教师分析代替学生阅读；问题碎片化割裂文本整体感；缺少从读到写的迁移。

---

## 讨论课

**核心关注**：讨论规则、参与广度、观点碰撞、结论建构。

| 特化维度 | 观察要点 |
|---|---|
| 议题设计 | 议题是否有讨论价值？是否开放且聚焦？ |
| 规则建立 | 是否有明确的讨论规则（轮流发言、倾听、回应）？ |
| 参与广度 | 是否所有学生都参与了？是否有"沉默者"？ |
| 观点碰撞 | 是否出现了不同观点的交锋？如何处理分歧？ |
| 深度推进 | 讨论是否从表面走向深入？ |
| 结论建构 | 结论是学生讨论的产物还是教师的总结？ |

**易犯错误**：议题太封闭没有讨论空间；变成少数学生的表演；讨论浮于表面缺乏深度。

---

## 体育课

**核心关注**：运动技能习得、练习密度、安全保障、个体差异。

| 特化维度 | 观察要点 |
|---|---|
| 热身设计 | 热身是否与主教材关联？是否充分？ |
| 技能教学 | 示范是否规范？分解是否合理？ |
| 练习密度 | 学生实际运动时间占比是多少？等待时间是否过长？ |
| 个体差异 | 是否关注不同体能水平的学生？是否有分层要求？ |
| 安全保障 | 场地器材检查、安全提示、保护帮助是否到位？ |
| 评价反馈 | 是否有即时的技术动作反馈？ |

**易犯错误**：讲解过多挤压练习时间；练习密度低；安全意识薄弱；缺少个体化指导。

---

## 美术/音乐课

**核心关注**：审美体验、创作过程、个性表达、技法与创意的平衡。

| 特化维度 | 观察要点 |
|---|---|
| 审美感受 | 是否引导学生感受美、发现美？ |
| 创作过程 | 是否有充分的创作/表现时间？ |
| 个性表达 | 是否鼓励个性化表达而非统一模板？ |
| 技法指导 | 技法指导是否服务于表达而非成为目的？ |
| 展示评价 | 是否有展示和互评环节？评价标准是否关注审美和创意？ |

**易犯错误**：过度强调技法训练而忽略审美体验；作品评价标准单一；缺少展示和交流环节。

---

## 项目式/探究式学习课

**核心关注**：问题驱动、自主探究、合作学习、成果展示。

| 特化维度 | 观察要点 |
|---|---|
| 驱动问题 | 问题是否有真实性和挑战性？ |
| 方案设计 | 学生是否参与了方案设计？ |
| 自主探究 | 学生是否有充分的自主探究时间？ |
| 合作质量 | 合作是否有明确分工？是否真正协同？ |
| 成果展示 | 展示形式是否多样？是否有反思和评价？ |
| 过程指导 | 教师何时介入指导？是否过度干预？ |

**易犯错误**：问题缺乏挑战性；自主探究时间不足；合作流于形式；教师过度干预或放任不管。
