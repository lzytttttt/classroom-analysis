# 课堂分析示例：输入与输出

本文档展示一个完整的课堂分析示例。先给出课堂实录输入，再给出对应的分析报告输出（第 1-6 段）。

---

## 输入：课堂实录片段

> **学科**：初中数学 | **课题**：等腰三角形的性质 | **课型**：新授课

```
【00:00-02:30】导入

师：同学们，我们之前学过三角形的分类，谁能说说按边怎么分类？
生1：等边三角形和不等边三角形。
师：还有吗？（停顿）等边三角形是一种特殊的三角形，那介于等边和不等边之间，有没有一种情况？
生2：两条边相等的三角形。
师：对，两条边相等的三角形，我们叫它——
全班：等腰三角形。
师：好，今天我们来研究等腰三角形有什么性质。（板书课题）

【02:30-08:00】探究活动

师：请同学们拿出课前准备好的等腰三角形纸片，沿顶角的角平分线对折，看看你发现了什么。
（学生动手操作，约2分钟）
师：谁来说说你发现了什么？
生3：折起来以后，两部分完全重合了。
师：重合说明了什么？
生3：说明两个底角相等。
师：很好。还有吗？（等待）折痕和底边有什么关系？
生4：折痕垂直于底边。
师：也就是说，折痕是底边的——
生4：垂直平分线。
师：等一下，我们再看看，折痕是不是平分了底边？大家再折一次验证一下。
（学生再次操作）
生4：哦，不一定是平分底边，但一定垂直。
师：对，这里要小心。折痕是顶角的平分线，它垂直于底边，但不一定平分底边。所以我们能得到什么结论？
生5：等腰三角形的两个底角相等。
师：这就是等腰三角形的第一个性质——等边对等角。（板书）

【08:00-13:00】证明与应用

师：刚才我们通过折纸发现了这个性质，但在数学中，发现不等于证明。谁能用我们学过的全等三角形知识来证明"等边对等角"？
（学生思考，约1分钟）
生6：可以做底边上的中线，然后用SSS证明两个三角形全等。
师：为什么做中线就能用SSS？
生6：因为中线把底边分成两段相等的线段，加上两条腰相等，还有公共边，三边对应相等。
师：很好。还有其他方法吗？
生7：做顶角的平分线，用SAS。
师：为什么是SAS？
生7：角平分线分出来的两个角相等，两条腰相等，还有公共边……不对，公共边不是夹角。
师：你再想想，SAS的条件是什么？
生7：两条边和它们的夹角。角平分线分出来的两个角是夹角，两条腰是夹边，公共边……
师：公共边是不是夹在两个角之间的边？
生7：对！所以是SAS。
师：很好。还有第三种方法吗？
生8：做底边上的高，用HL。
师：为什么？
生8：高把底边分成两段，因为等腰三角形的高也是底边的中线……
师：等一下，你怎么知道高就是中线？
生8：因为……可以证明两个直角三角形全等。
师：这个思路对，但需要先证明。我们把它留作课后思考。

【13:00-16:00】练习

师：现在看这道题。（展示题目：等腰三角形中，一个角是70度，求其他两个角的度数。）
（学生练习，约2分钟）
师：谁来说？
生9：如果70度是顶角，那两个底角各55度。如果70度是底角，那另一个底角也是70度，顶角是40度。
师：为什么要分两种情况？
生9：因为题目没说70度是顶角还是底角。
师：很好，这就是分类讨论的思想。（板书两种情况）
师：如果把70度改成100度呢？
生10：100度只能是顶角，因为如果是底角，两个底角就200度了，超过180度了。
师：非常好。这说明什么？
生10：等腰三角形的底角必须是锐角。
师：为什么？
生10：因为两个底角相等，如果有一个是直角或钝角，三个角加起来就超过180度了。
```

#### 3.2 关键教学行为解读

**片段1：追问促自我修正（04:00-05:00）**

- 证据：生4说"折痕垂直于底边"，教师追问"折痕是不是平分了底边？大家再折一次验证"，生4修正为"不一定是平分底边，但一定垂直"
- 分析：教师没有直接纠正生4的错误（如果直接说"不对"），而是通过追问引导验证，让学生自己发现错误并修正。这保护了学生的思维安全感，同时培养了验证意识
- 教研价值：追问策略在学生出现不严谨表述时的有效运用，体现了"让学生自己发现错误"的教学理念

**片段2：追问促深层理解（10:30-11:30）**

- 证据：生7用SAS证明时说"公共边……不对，公共边不是夹角"，教师追问"SAS的条件是什么？""公共边是不是夹在两个角之间的边？"，生7最终确认是SAS
- 分析：教师没有直接告诉答案，而是通过分解问题帮助学生自我排查错误。追问的指向性很明确（回到SAS定义），避免了学生在错误方向上越走越远
- 教研价值：证明教学中"回到定义"的追问策略，比直接纠正更有效

#### 5.2 互动反馈诊断

- **能够接住学生回答的片段**：
  - 生4说"垂直"→教师追问"是不是平分"→学生验证后自我修正（04:00-05:00）

- **未充分展开学生回答的片段**：

- **问题链的递进性判断**：
  - 整体呈现清晰的递进结构：回忆（记忆）→ 操作发现（理解）→ 证明（分析/应用）→ 变式练习（应用/分析/评价）

- **可改进的追问示例**：

### 6. 认知层级分析

| 环节/任务 | 主要认知层级 | 依据 | 是否出现层级递进 | 可提升空间 |
